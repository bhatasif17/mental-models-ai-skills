# B2B Insight Seller

## Purpose
B2B Insight Seller helps users with this problem: Strong B2B sales often reframes the buyer problem.

This is an original AI workflow inspired by the general problem space of **The Challenger Sale**. It is not a summary, replacement, derivative copy, or reproduction of the book.

## When to use this skill
Use this skill when the user wants to: create B2B sales messages that teach buyers a useful insight instead of listing services.

## Required user inputs
Prefer to collect these inputs, but proceed with sensible assumptions if the user already gave enough context:
- service
- target buyer
- buyer problem
- market trend
- offer


## Operating workflow
1. Restate the user's situation in one or two sentences.
2. Diagnose the real problem behind the request.
3. Identify constraints, risks, and missing information.
4. Build a practical strategy tailored to the user's context.
5. Produce ready-to-use outputs, not generic advice.
6. Add mistakes to avoid.
7. End with one useful next action.

## Output format
Return the answer using these sections:

### Diagnosis
Explain what is really going on.

### Strategy
Give the recommended approach.

### Ready-to-use output
Produce the user's requested asset.

### Mistakes to avoid
List the most important risks or traps.

### Next action
Give one concrete next step.

## Expected outputs
- commercial insight
- problem reframe
- cold message
- call opener
- follow-up email
